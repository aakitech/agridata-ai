# Archive

**Status:** Historical

Archived docs are retained for context but are not the current source of truth.

Use this folder for:

- old specs
- completed changelogs
- historical roadmaps
- one-off audits
- implementation plans that have already served their purpose

If an archived document becomes relevant again, promote the useful parts into `docs/architecture`, `docs/adr`, or `docs/product` rather than treating the old document as current.
